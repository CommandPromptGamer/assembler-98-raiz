
Passos para build : Dependencias GTK4, cmake, make 
- mkdir build
- cd build
- cmake ..
- make
- ./A9R

Em sistemas UNIX. 

Instalando GTK4 no ubuntu : 
- sudo apt update
- sudo apt install libgtk-4-dev

ou 
- sudo add-apt-repository ppa:gnome3-team/gnome3-staging
- sudo apt update
- sudo apt install libgtk-4-dev
